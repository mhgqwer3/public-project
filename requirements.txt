# Smart Bins System - Development Requirements
# PHP والخوادم المطلوبة

## Server Requirements
- PHP >= 8.2
- MySQL/MariaDB >= 10.4 (or MySQL >= 5.7)
- Apache/Nginx Web Server

## PHP Extensions Required
- php-pdo (PDO Database Abstraction)
- php-pdo-mysql (MySQL Driver for PDO)
- php-json (JSON support)
- php-curl (cURL support)
- php-filter (Input validation)

## Frontend Libraries (included in project)
- Bootstrap 5.x (CSS Framework)
- Font Awesome (Icons)
- JavaScript ES6+

## Database
- Database: smart_bins_system
- SQL file: Database/smart_bins_system.sql
- Character set: utf8mb4

## Configuration Files
- .env (Environment variables configuration)

## Docker Requirements (Optional)
- Docker >= 20.x
- Docker Compose >= 2.x
